# Kiro Configuration

このディレクトリには、Kiro AI エージェントの設定とステアリングファイルが含まれています。

## グローバル設定

このプロジェクトは、以下のグローバルベストプラクティスを使用しています：

**Repository**: [kumagaias/kiro-best-practices](https://github.com/kumagaias/kiro-best-practices)

グローバル設定には以下が含まれます：
- 共通の技術標準
- プロジェクト標準
- コーディング規約
- セキュリティベストプラクティス

## プロジェクト固有の設定

### Steering Files (`steering/`)

プロジェクト固有のガイダンスファイル：
- `tech.md` - TriageNinja の技術アーキテクチャ
- `project.md` - プロジェクト標準とワークフロー

### Settings (`settings/`)

- `mcp.json` - Model Context Protocol (MCP) サーバー設定

## 詳細

グローバル設定の詳細については、[kiro-best-practices](https://github.com/kumagaias/kiro-best-practices) リポジトリを参照してください。
