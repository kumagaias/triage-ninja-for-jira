# Screenshots

This directory contains screenshots for the README.md file.

## Required Screenshots

1. **dashboard.png** - Dashboard view with statistics and ticket list
2. **triage-panel.png** - AI Triage panel on issue detail page
3. **triage-results.png** - AI analysis results with recommendations

## How to Capture Screenshots

1. Open your Jira instance with TriageNinja installed
2. Navigate to the TriageNinja Dashboard
3. Take a screenshot (1920x1080 or higher resolution)
4. Save as `dashboard.png`
5. Open any issue with the AI Triage panel
6. Take a screenshot and save as `triage-panel.png`
7. Run AI Triage and capture the results
8. Save as `triage-results.png`

## Tips

- Use high resolution (1920x1080 or higher)
- Ensure UI is clean and professional
- Use real data (not test data)
- Crop to show relevant parts only
- Optimize file size (< 500KB per image)
